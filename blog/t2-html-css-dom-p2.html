What is grid based design?

Grid based design is a technique to create and manage easy, scalable web content. The basic premise involves creating 12 classes in HTML, titled col-1, col-2 etc. These columns take up a certain percentage of the screen, regardless of the screen size. By doing this, you can easily slot content into “boxes” on the grid by defining them as a certain size class. For example, if you want something that will take up exactly half of your screen, regardless of web browsing platform, you would set it within a size class of <div class=“col-6”>. And, because in your CSS document you have specified that .col-6 has a width: element of 50%, the content within your <div> will scale accordingly.

       

Whats all the hype about responsive webpages?

Come on man, we don’t really need to have this conversation again, do we?
